1. Sign up github.com. 建立本學期作業的repository.

2. 在你的local 硬碟中建立一個本學期作業的資料夾。並在資料夾內建立index.html，作為本學期作品集的title page，格式可參看這個範例。每次作業有一格animated gif以及url link.

3. 實作Xmas Light，技術要求：

  • object-oriented: private/public properties, public methods, private functions
  • name: a string to identify this light
  • power on/off; when power is on, start blinking
  • every light is associated with a button to control on/off
  • every light is in different color: 以hue設定 (reference)
  • 用建構子建立不只一個Light

4. 使用jsfiddle（先sign up）

5. 再轉成html （將先前的jsfiddle以hyperlink列在標題中)，如hw1demo。

6. 將hw1加入index.html中。
