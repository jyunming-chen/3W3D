Homework 5:

File system as Database, node.js, shelljs, express

Due: 5/31/2017

技術重點：

  • 利用檔案系統實作資料庫
  • 利用Ｃ程式操作檔案：開/關 檔案，查詢/插入 資料

Sign In/Sign Up畫面

  • 鍵入 login ID (one word)。若使用者存在，則做Sign In
  • 若不存在，則先做Sign Up (加入），再Sign In

Logging

  • Sign In後轉入Logging畫面(Start loggin)，以當時時間為start time，一律從location 0開始。
  • 每當進入下一個Location 時：按下前一列末的“next", 下一列的Time及Location的<input>會自動出現，Time並填入當下時間。鍵入 location code (正整數）。
  • Logging 區塊 <div>若超過所給高度，則會有捲軸出現（scrollable)

Saving

  • 按下”Save Log"結束logging。將結果寫入 [loginID]YYYYMMDD.log,
  • 按下”Check Log"，讀回該檔案，將內容印製html中，確認內容與剛剛的內容相符。

Submission

  • 將所有過程，以 animated gif 記錄於 index.html中。
